Was kann man im Rechner-Screen machen?

Man kann eine Vielzahl an Ausdrücken numeriscb berechnen. Dabei wurden elementare Funktionen, Integrale und Ableitungen implementiert. Es gibt aber auch Zahlentheoretische Funktionen und Lösungen zu analytischen Aufgaben (mehr dazu im Analysis-Screen).

Hier eine Liste der möglichen Befehle:

Funktionen:

Ausdruck Funktion Beispiel
cos, arccos, cosh, arccosh,
sin, arcsin, sinh, arcsinh,
tan, arctan, tanh, arctanh		 Elementare trigonometrischen Funktionen cos(pi)
exp(x) oder e^x Exponentialfunktion e^4
pow(a,b) oder a^b potenzen (-2.4)^5
sqrt(x) Wurzelfunktion sqrt(2)
ln(x) natürlicher Logarithmus ln(2)
log(x,n) Logarithmus (Basis n) von x log(10,2)

Differentialoperatoren:

Ausdruck Funktion Beispiel
d/dx(f(x)) Bestimmt die exakte Ableitung der Funktion f. Dabei kann f von weiteren Parametern abhängen. d/dx(log(a*x^2-b))
d^n/dx^n(f(x))(x_0) bestimmt die approximative n-te Ableitung von f in x_0. Kann instabil sein. d^3/dx^3(sin(x))(1)
Int(a,b,f(x),x) Integral von f(x)dx von a nach b.
Rechts von der Eingabeleiste werden die Approximationsverfahren vorgeschlagen.
Es wird eine obere Schranke für den Fehler angegeben 		Int(2,3,sin(x)^2,x)

Zahlentheorie:

Ausdruck Funktion Beispiel
C(a,b) oder aCb Binomialkoeffizienten 9C4
fact(n) oder n! Faktoriell 5!
kgV(a,b) oder PDCM(a,b) kleinstes gemeinsames Vielfache von a und b kgV(4,22)
ggT(a,b) oder PGCD(a,b) größter gemeinsame Teiler von a und b ggT(45,25)
isprime(n) Falls n eine Primzahl ist, wird "True" ausgegeben. Sonst "False". Es wird dabei nach Teilern gesucht. isprime(527)
eratosthenes(n) Gibt eine Liste von den Primzahlen kleiner als n aus. eratosthenes(100)
primfactors(n) Gibt die Primteilerzerlegung der Zahl n an. primfactors(137435)
partition(n) Partitionszahl von n partition(23)
